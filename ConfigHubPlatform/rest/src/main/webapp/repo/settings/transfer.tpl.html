What happens to the existing repository teams and members?

They move with the repository. Their access to the repository is unchanged.


What happens to the current repository owners and administrators?

They will no longer be associated to the repository. If the new owner is an organization, then the new owners of that organization can add current owners to their organization.


What happens to repository tokens?

They stay the same and will remain valid. Clients do not need to change, and can continue to pull configuration as they currently do.
